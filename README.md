# RFID-Attendance-with-FRAM-EEPROM

## Smart RFID-Based Attendance System Using ESP32

ESP32-based RFID attendance system with FRAM and EEPROM storage for secure and reliable attendance management.

## Overview
This project is an IoT-based smart attendance system that uses RFID technology to identify users and stores attendance records using FRAM and EEPROM memory.

## Features
- RFID card-based identification
- ESP32 microcontroller control
- FRAM and EEPROM data storage
- Secure attendance record management
- Reliable embedded system design

## Hardware Components
- ESP32 Development Board
- RFID Reader Module
- RFID Cards/Tags
- FRAM Memory Module
- EEPROM Memory Module
- LCD Display
- Jumper Wires

## Working
The RFID reader scans the user's RFID card. The ESP32 processes the card information and stores attendance data in FRAM and EEPROM memory for future retrieval.

## Applications
- Smart classrooms
- Office attendance systems
- Access control systems
